1. `10950 번 : A+B - 3` : 
    두 정수 `A`, `B` 의 합을 출력하는 문제

2. `2609 번 : 최대공약수와 최소공배수` : 
    주어진 두 수의 최대공약수와 최소공배수를 출력하는 문제

3. `5622 번 : 다이얼` : 
    다이얼 전화를 돌리는데 걸리는 시간을 출력하는 문제

4. `11401 번 : 이항 계수 3` : 
    주어진 이항 계수를 계산하는 문제
